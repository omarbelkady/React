## Lists in ReactJS 


### Empty Circle List Icon
```css
ul{
    list-style-type: circle;
}
```

### Filled Square List Icon
```css
ul{
    list-style-type: square;
}
```

### Numbered List Icon
```css
ul{
    list-style-type: decimal;
}
```

### ABC List Icon in caps locks
```css
ul{
    list-style-type: upper-alpha;
}
```
