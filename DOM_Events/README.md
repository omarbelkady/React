## DOM Events In React
#### Most Essential Events in React
1. onClick is an event handler used to handle click events usually for buttons
2. onChange is an event handler used to handle keyboard events

### Mouse Evs
- onclick
- onmouseover
- onmouseout

### Form Evs
- onchange
- onfocus
- onblur
- oninput


### Keyboard Evs
- onkeydown
- onkeyup
- onkeypress
